Please find below some explanations and step by steps to run the URL shortener tool.
- Checkout the repository on your local machine
- Open a command line in the same folder as 'package.json'
- Run 'npm install' to fetch all the node modules needed for this tool
- Once it is done, run 'node server.js' to start the server
- Open a browser and go to 'http://localhost:8080/index' to access the web page for the url shortener
- Copy paste a URL in the field, and click OK to get your shortened URL
- The shortened URL will redirected automatically to the long URL, even if you stop and restart the server
